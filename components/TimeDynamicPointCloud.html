TimeDynamicPointCloud

TimeDynamicPointCloud is a point cloud with dynamic changes with time.

  • Cesium element: TimeDynamicPointCloud
  • Example: TimeDynamicPointCloud

Availability#

Inside Viewer or CesiumWidget component. A TimeDynamicPointCloud object will be attached to the PrimitiveCollection of the Viewer or CesiumWidget.
